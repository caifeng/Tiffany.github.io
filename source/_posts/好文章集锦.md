### Sass

Sass 是一种 CSS 的预编译语言，Sass 为 CSS 赋予了更强大的功能。

它提供了 变量（variables）、嵌套（nested rules） [混合（mixins）、函数（functions）等功能，并且完全兼容 CSS 语法。

Sass 能够帮助复杂的样式表更有条理， 并且易于在项目内部或跨项目共享设计。
